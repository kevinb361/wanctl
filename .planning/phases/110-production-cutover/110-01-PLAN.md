   ```yaml
   cake_params:
     download_interface: "ens17"    # Spectrum router-side NIC (CAKE target)
     upload_interface: "ens17"      # Same NIC -- both directions shaped on egress
     overhead: "docsis"             # Spectrum cable (per CAKE-05)
     memlimit: "32mb"              # Per CAKE-06
     rtt: "100ms"                  # Per CAKE-10 (tunable, conservative default)
   ```
   Note: Both download_interface and upload_interface are ens17 -- CAKE creates separate download/upload qdiscs on the same router-side bridge member port. The direction is handled by build_cake_params() adding `ingress` for download and `ack-filter` for upload.
