main() should become a ~30-40 line orchestrator with this structure:
```
def main():
    args = _parse_autorate_args()
    if args.validate_config: return validate_config_mode(args.config)
    controller = ContinuousAutoRate(args.config, debug=args.debug)
    _configure_profiling(controller, args)
    _configure_dry_run(controller, args)
    lock_files = _acquire_daemon_locks(controller)
    _setup_signal_handlers(controller)
    irtt_thread = _start_irtt_thread(...)
    metrics_server, health_server = _start_servers(...)
    try:
        _run_daemon_loop(controller, irtt_thread, ...)
    finally:
        _cleanup_daemon(controller, lock_files, irtt_thread, metrics_server, health_server)
```

2. `_configure_dry_run(controller, args)` -- lines ~495-504 (set dry_run on WANControllers if --dry-run flag). Small, can combine with profiling.

3. `_setup_signal_handlers(controller)` -- register SIGTERM/SIGINT/SIGUSR1 handlers (lines ~506-530ish). This block may already be inline or partially extracted.

4. `_run_daemon_loop(controller, irtt_thread, metrics_server, health_server, ...)` -- the entire while-not-shutdown loop body (lines ~560-935). This is ~375 lines and ITSELF needs sub-extraction:

   4a. `_run_control_cycles(controller)` -- the inner for loop that calls run_cycle() on each WAN controller + watchdog notification (lines ~570-620ish)
   
   4b. `_run_maintenance_tasks(controller, ...)` -- hourly maintenance: retention cleanup, downsampling, vacuum (lines ~630-680ish)
   
   4c. `_run_adaptive_tuning(controller)` -- the entire adaptive tuning block (lines ~680-901). This is ~220 lines and is the biggest sub-block. Extract as a single function that handles:

   5c. `_release_daemon_locks(controller, lock_files)` -- release lock files
   5d. `_close_router_connections(controller, deadline, logger)` -- close SSH/REST connections
   5e. `_stop_daemon_servers(controller, metrics_server, health_server, deadline, logger)` -- stop metrics + health servers
   5f. `_close_metrics_writer(deadline, logger)` -- close MetricsWriter
