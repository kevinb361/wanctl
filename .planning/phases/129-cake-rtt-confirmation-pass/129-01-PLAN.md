## Test Methodology (inherited from Phases 127-128, per D-01)

1. Set value A in YAML, SIGUSR1 reload, wait 5s
2. Run `flent rrul_be -H 104.200.21.31 -l 60`, note time of day
3. Record: ICMP median, p99, max; DL throughput; UL throughput
4. Set value B, SIGUSR1 reload, wait 5s
5. Run same flent, note time of day
6. Compare metrics. If delta < 5%, re-run both (per D-02)
7. Apply winner

## Confirmation Pass Logic (per D-06, D-07, D-08)

Each confirmation test: current winner (A) vs original loser (B), full winner set active.
If winner FLIPS (original loser now wins), KEEP THE NEW WINNER -- full-set result is authoritative.
