<objective>
Close the v1.40 silent-ignore gap (D-08, SAFE-06) by emitting a WARNING log entry on every daemon startup for any `continuous_monitoring.*` key (or any other path) not present in `KNOWN_AUTORATE_PATHS`. Add a regression test that exercises the unknown-key path.

Purpose: Production `/etc/wanctl/spectrum.yaml` carried 4 unrecognized UL keys (`factor_down_yellow: 0.98`, `target_bloat_ms: 42`, `warn_bloat_ms: 105`, `ceiling_mbps: 18`) for 3 days starting 2026-04-29 with no startup warning. The daemon was running half-shipped (YAML had keys, binary did not have code). The offline `wanctl-check-config` CLI does detect unknown keys via `check_unknown_keys`, but operators don't run that as a deployment gate — startup-time emission is the correct surface.

Output: One added emission helper in `Config.__init__` (or `_load_specific_fields`), one regression test asserting the WARNING is emitted with the unknown-key path in the message.
</objective>

    ```python
        def _warn_unknown_continuous_monitoring_keys(self) -> None:
            """Emit a WARNING log entry for any leaf key in self.data not in KNOWN_AUTORATE_PATHS.

            Closes the SAFE-06 silent-ignore gap. Operators changing config in
            production must see drift between YAML and the deployed code surface.
            Hard-reject was rejected (D-08): too disruptive for legitimate cases
            where new keys ship in YAML before the matching daemon. WARNING is
            audible without blocking startup.
            """
            # Local import — avoid circular import at module load time
            # (check_config_validators imports Config).
            from wanctl.check_config_validators import KNOWN_AUTORATE_PATHS, _walk_leaf_paths

            logger = logging.getLogger(__name__)
            try:
                all_paths = _walk_leaf_paths(self.data)
            except Exception as e:  # never abort startup over a path-walk bug
                logger.warning(
                    f"Unknown-key check skipped due to walker error: {e}"
                )
                return

            for path in all_paths:
                # Mirror check_unknown_keys: skip dynamic alerting.rules.* sub-keys.
                if path.startswith("alerting.rules."):
                    continue
                if path not in KNOWN_AUTORATE_PATHS:
                    logger.warning(
                        f"Unknown config key in {self.config_file_path}: {path} "
                        f"(SAFE-06: ignored by daemon — verify your YAML matches the deployed binary)"
                    )
    ```

                rec for rec in caplog.records
                if rec.levelno == logging.WARNING
                and "target_bloat_ms_typo" in rec.getMessage()

                f"{[r.getMessage() for r in caplog.records]}"
