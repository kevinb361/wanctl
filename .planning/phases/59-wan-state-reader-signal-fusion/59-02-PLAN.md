<objective>
Extend BaselineLoader to extract WAN congestion zone from the same state file read used for baseline RTT, with 5-second staleness fail-safe, and wire the zone through to ConfidenceSignals in update_state_machine().

Purpose: FUSE-01 mandates zero additional I/O -- zone extraction piggybacks on the existing safe_json_load_file() call. SAFE-01 ensures stale data defaults to GREEN rather than influencing steering decisions with outdated congestion information.

Output: BaselineLoader returns (baseline_rtt, wan_zone) tuple, SteeringDaemon stores and threads wan_zone to confidence scoring.
</objective>
