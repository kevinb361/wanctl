       ```
           with Vertical(id="wan-col-2", classes="wan-col"):

       ```

    7. Add `_apply_layout(self)` method: read `self.size.width`, compute new_mode ("wide" if >= BREAKPOINT_WIDE else "narrow"), skip if same as `_layout_mode`, otherwise update `_layout_mode` and toggle CSS classes on `#wan-row` via `wan_row.set_class(new_mode == "wide", "wide-layout")` and `wan_row.set_class(new_mode == "narrow", "narrow-layout")`.

    ```css
    }
    ```
