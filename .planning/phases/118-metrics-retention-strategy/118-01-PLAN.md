   ```python
   def validate_retention_tuner_compat(

       logger: logging.Logger | None = None,
   ) -> None:
   ```
   Logic:
   - If `tuning_config is None` or `not tuning_config.get("enabled", False)`: return immediately (no constraint).
   - Extract `lookback_hours = tuning_config.get("lookback_hours", 24)`, compute `lookback_seconds = lookback_hours * 3600`.
