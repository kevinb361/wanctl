```bash
run_flent() {
    local label="$1"
    local run_num="$2"
    output=$(flent rrul -H "$FLENT_HOST" -l "$FLENT_DURATION" -t "${label}-run${run_num}" 2>&1)
    icmp_median=$(echo "$output" | grep "Ping (ms) ICMP" | awk '{print $6}')
    icmp_p99=$(echo "$output" | grep "Ping (ms) ICMP" | awk '{print $7}')
    dl_sum=$(echo "$output" | grep "TCP download sum" | awk '{print $5}')
    ul_sum=$(echo "$output" | grep "TCP upload sum" | awk '{print $5}')
}
```

   - Thresholds: `TCP12_P99_THRESHOLD=500` (VAL-01), `REGRESSION_TOLERANCE=10` (percent, VAL-02)

3. **Argument parsing:** Parse `--test`, `--runs`, `--duration` with case statements in a while loop. Default `--test all`.

4. **Pre-flight gate (mandatory):**
   - Run `bash scripts/check-tuning-gate.sh` -- exit 1 on failure.
   - Query health endpoint to verify burst detection is active: `ssh $VM_HOST 'curl -s http://127.0.0.1:9101/health'` piped to `jq '.wans[0].burst_detection.enabled'` and `jq '.wans[0].burst_detection.burst_response_enabled'`. Both must be `true` or exit with error.

5. **run_flent_test() function:** Accept test_name, label, run_num. Execute `flent "$test_name" -H "$FLENT_HOST" -l "$DURATION" -t "${label}-run${run_num}" 2>&1`. 6. **run_test_suite() function:** Accept test_type. Loop `$RUNS` times, call run_flent_test, collect p99 values into an array. After all runs, compute average p99 using awk. For tcp_12down: compare avg p99 against `$TCP12_P99_THRESHOLD`. For rrul/rrul_be: compute delta percent from baseline, compare against `$REGRESSION_TOLERANCE`. Print per-run table and verdict (PASS/FAIL).

7. **Main logic:** Based on `--test` argument, call run_test_suite for each selected test type. Print summary at end with all verdicts.

8. **Output format:** Print a structured results table to stdout. Use colors (GREEN/RED) for PASS/FAIL matching the check-tuning-gate.sh pattern. Include timestamp, version, environment details at top.

- Do NOT use flent `-H dallas` -- use the IP directly.
- Match shell style of cake-param-sweep.sh: `set -euo pipefail`, color constants, log functions.
- Make the script executable: end with `chmod +x` note in comments.
