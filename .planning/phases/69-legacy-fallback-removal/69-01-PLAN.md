1. Add `deprecate_param()` to config_validation_utils.py:
   ```python
   def deprecate_param(
       config: dict,
       old_key: str,
       new_key: str,
       logger: logging.Logger,
       transform_fn: typing.Callable | None = None,
   ) -> typing.Any | None:
   ```
   - If old_key not in config, return None
   - If old_key in config and new_key also in config, return None (modern takes precedence, no warning)
   - If old_key in config and new_key NOT in config: log warning, apply transform_fn if provided, return translated value
   - Warning format: `"Deprecated config parameter '{old_key}': use '{new_key}' instead (value translated: {translated_value})"`

2. Wire into autorate_continuous.py `_load_threshold_config()`:
   - Before the existing if/elif/else chain for baseline, call deprecate_param on thresh dict with old_key="alpha_baseline", new_key="baseline_time_constant_sec", transform_fn that converts alpha to time_constant (cycle_interval/alpha)
   - If deprecate_param returns a value, inject it into thresh["baseline_time_constant_sec"] so the existing if-chain picks it up naturally
   - Same pattern for alpha_load -> load_time_constant_sec
   - This preserves the existing if/elif/else structure with minimal code change

3. Wire into steering/daemon.py `_load_state_sources()`:
   - Replace the manual spectrum->primary fallback with deprecate_param call on self.data["cake_state_sources"] dict
   - If deprecate_param returns a value, set self.primary_state_file from it

4. Wire into steering/daemon.py `_load_cake_queues()`:
   - Replace the nested .get("spectrum_download") fallback with deprecate_param call
   - Same for spectrum_upload -> primary_upload
