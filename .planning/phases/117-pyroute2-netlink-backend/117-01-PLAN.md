<objective>
Create NetlinkCakeBackend class that replaces subprocess tc with pyroute2 netlink for CAKE bandwidth changes, with singleton IPRoute lifecycle management and per-call subprocess fallback.

Purpose: Reduce hot-loop tc call latency from 3.1ms (subprocess fork/exec) to <0.5ms (netlink), reclaiming ~5ms per 50ms control cycle. The subprocess fallback guarantees zero service disruption if netlink fails.

Output: `netlink_cake.py` with NetlinkCakeBackend class + comprehensive tests + pyroute2 dependency in pyproject.toml.
</objective>

    **Step 2: Create `src/wanctl/backends/netlink_cake.py`.**

    Class `NetlinkCakeBackend(LinuxCakeBackend)` with:

    ```python
    import logging
    from typing import Any

    from wanctl.backends.linux_cake import LinuxCakeBackend, OVERHEAD_KEYWORD_EXPANSION

    # Lazy import to avoid hard dependency
    # pyroute2 is optional -- only needed for linux-cake-netlink transport
    _pyroute2_available = False
    try:
        from pyroute2 import IPRoute
        from pyroute2.netlink.exceptions import NetlinkError
        _pyroute2_available = True
    except ImportError:
        IPRoute = None  # type: ignore[misc,assignment]
        NetlinkError = OSError  # type: ignore[misc,assignment]
    ```

    Constructor: `__init__(self, interface, logger=None, tc_timeout=5.0)` -- calls super().__init__(), sets `self._ipr: IPRoute | None = None`, `self._ifindex: int | None = None`.

    `_get_ipr(self) -> IPRoute`:
    - If `not _pyroute2_available`: raise ImportError("pyroute2 not installed")
    - If `self._ipr is None`: create `IPRoute(groups=0)`, call `link_lookup(ifname=self.interface)`, raise OSError if empty, store ifindex
    - Return self._ipr

    `set_bandwidth(self, queue: str, rate_bps: int) -> bool`:
    - Convert rate_bps to kbit: `rate_kbit = rate_bps // 1000`
    - Try: `ipr = self._get_ipr()`, `ipr.tc("change", kind="cake", index=self._ifindex, bandwidth=f"{rate_kbit}kbit")`, log debug, return True
    - Except (NetlinkError, OSError, ImportError): log WARNING with "falling back to subprocess", set `self._ipr = None`, return `super().set_bandwidth(queue, rate_bps)`

    `get_bandwidth(self, queue: str) -> int | None`:
    - Try: `ipr = self._get_ipr()`, `msgs = ipr.tc("dump", index=self._ifindex)`, find CAKE msg, extract bandwidth from options TCA_CAKE_BASE_RATE64, convert from bytes/sec to bps (multiply by 8), return int
    - Except: log WARNING, `self._ipr = None`, return `super().get_bandwidth(queue)`

    `initialize_cake(self, params: dict[str, Any]) -> bool`:
    - Try: build pyroute2 kwargs from params dict:
      - `bandwidth` -> `bandwidth=str(params["bandwidth"])` (already in "Nkbit" format)
      - `diffserv` -> `diffserv_mode=str(params["diffserv"])` (e.g., "diffserv4")
      - `overhead_keyword` -> map: "docsis" becomes `overhead=-1`; OVERHEAD_KEYWORD_EXPANSION entries need special handling (e.g., "bridged-ptm" -> `atm_mode="ptm", overhead=22`); other keywords (like "conservative") -> pyroute2 may not support directly, fall through to subprocess
      - `overhead` (numeric) -> `overhead=int(params["overhead"])`
      - `mpu` -> `mpu=int(params["mpu"])`
      - `memlimit` -> `memlimit=int(params["memlimit"])`
      - `rtt` -> `rtt=str(params["rtt"])` (pyroute2 handles "100ms", "internet" presets)
      - `split-gso` -> `split_gso=True`
      - `ack-filter` -> `ack_filter=True`
      - `ingress` -> `ingress=True`
    - Call `ipr.tc("replace", kind="cake", index=self._ifindex, **kwargs)`
    - Log info, return True
    - Except: log WARNING, `self._ipr = None`, return `super().initialize_cake(params)`

    `validate_cake(self, expected: dict[str, Any]) -> bool`:
    - Try: `msgs = ipr.tc("dump", index=self._ifindex)`, find CAKE msg, read back options via TCA attrs, compare against expected
    - Except: fallback to `super().validate_cake(expected)`

    `test_connection(self) -> bool`:
    - Try netlink path first: `_get_ipr()`, `tc("dump", index=self._ifindex)`, check for CAKE kind
    - Except: fallback to `super().test_connection()`

    `close(self)`:
    - If self._ipr is not None: try ipr.close() except pass, set self._ipr = None

    `from_config(cls, config, direction="download") -> "NetlinkCakeBackend"`:
    - Same logic as LinuxCakeBackend.from_config but returns NetlinkCakeBackend instance
    - Read interface from cake_params.{direction}_interface
    - Read tc_timeout from timeouts.tc_command (default 5.0)
