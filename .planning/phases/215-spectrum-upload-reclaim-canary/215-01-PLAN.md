|----------|-------------|
| deployed YAML (over SSH) → gate-script env preflight | gate trusts what the operator declares as expected ceiling; mismatch with deployed value must abort |
| flent artifact → extractor | untrusted/possibly-collapsed measurement data crosses into gate scoring |
| leg-A baseline inputs → derived thresholds | a drifted/wrong leg-A would skew the derived bounds; sanity-check vs static fallback flags large divergence |
| evidence artifacts → git | config/health snapshots may carry secret-shaped fields |
