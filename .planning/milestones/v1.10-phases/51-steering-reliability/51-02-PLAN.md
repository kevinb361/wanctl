<objective>
Improve BaselineLoader safety and staleness detection: replace raw open/json.load with safe_json_load_file (STEER-04) and add stale baseline detection with warning (STEER-03).

Purpose: BaselineLoader currently uses raw `open()`/`json.load()` with a manual try/except that is less robust than `safe_json_load_file()` which handles JSONDecodeError, OSError, and concurrent writes. Additionally, if autorate stops writing state (crash, hang), the steering daemon silently uses arbitrarily old baseline RTT data with no visibility.

Output: Updated BaselineLoader class with safe file loading and staleness checks, tests proving behavior.
</objective>
