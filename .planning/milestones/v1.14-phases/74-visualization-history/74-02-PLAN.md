    - HistoryBrowserWidget(Widget):
      - __init__(autorate_url: str, *, id: str | None = None)
      - Store autorate_url, create httpx.AsyncClient(timeout=5.0) lazily (not in __init__ to avoid issues before mount)

        1. Update summary Static to "Loading..."
        2. Create httpx.AsyncClient if not exists, fetch GET {autorate_url}/metrics/history?range={time_range}

        4. Clear DataTable, add_row for each record: (timestamp, wan_name, metric_name, f"{value:.2f}", granularity)
        5. Compute summary stats: group values by metric_name, for each group compute min/max/avg/p95/p99 using statistics module
        6. Format summary into Static widget
        7. On error: clear table, show "Failed to fetch" in summary
      - _compute_summary(values: list[float]) -> dict: use statistics.mean, statistics.quantiles for p95/p99, min/max builtins. Return {"min": ..., "max": ..., "avg": ..., "p95": ..., "p99": ...}. Handle empty and single-value edge cases.
