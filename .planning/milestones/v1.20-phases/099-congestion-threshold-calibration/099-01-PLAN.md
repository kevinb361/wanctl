    GREEN phase: Create src/wanctl/tuning/strategies/congestion_thresholds.py with:

    1. Module-level constants:
       - MIN_GREEN_SAMPLES = 60
       - STATE_GREEN = 0.0
       - DEFAULT_COV_THRESHOLD = 0.05
       - NUM_SUB_WINDOWS = 4

    2. _extract_green_deltas(metrics_data: list[dict]) -> list[float]:
       Build two dicts: state_by_ts (timestamp -> float) from rows where metric_name == "wanctl_state",
       delta_by_ts (timestamp -> float) from rows where metric_name == "wanctl_rtt_delta_ms".
       Return [delta_by_ts[ts] for ts in delta_by_ts if state_by_ts.get(ts) == STATE_GREEN].

    3. _extract_green_deltas_with_timestamps(metrics_data: list[dict]) -> tuple[list[float], list[int]]:
       Same logic as _extract_green_deltas but also returns the matched timestamps (needed for convergence sub-windowing).

    4. _is_converged(green_deltas: list[float], timestamps: list[int], percentile_index: int, cov_threshold: float = DEFAULT_COV_THRESHOLD) -> bool:
       Split timestamps into NUM_SUB_WINDOWS equal ranges. For each sub-window, collect deltas. If any sub-window has < 10 samples, return False. Compute statistics.quantiles(chunk, n=100)[percentile_index] for each sub-window. If fewer than 3 valid sub-percentiles, return False. Compute avg = mean(sub_percentiles). If avg < 0.001, return True. Compute cov = stdev(sub_percentiles) / avg. Return cov < cov_threshold.

    5. calibrate_target_bloat(metrics_data: list[dict], current_value: float, bounds: SafetyBounds, wan_name: str) -> TuningResult | None:
       - Call _extract_green_deltas_with_timestamps(metrics_data)
       - If len(green_deltas) < MIN_GREEN_SAMPLES: log INFO, return None
       - If _is_converged(green_deltas, timestamps, 74): log INFO "converged", return None
       - percentiles = quantiles(green_deltas, n=100)
       - candidate = percentiles[74]  # p75
       - confidence = min(1.0, len(green_deltas) / 1440.0)
       - Return TuningResult(parameter="target_bloat_ms", old_value=current_value, new_value=round(candidate, 1), confidence=confidence, rationale=f"p75 GREEN delta={candidate:.1f}ms ({len(green_deltas)} samples)", data_points=len(green_deltas), wan_name=wan_name)

    6. calibrate_warn_bloat(metrics_data: list[dict], current_value: float, bounds: SafetyBounds, wan_name: str) -> TuningResult | None:
       - Same pattern as calibrate_target_bloat but uses percentile_index 89 (p90)
       - parameter="warn_bloat_ms", rationale prefix "p90 GREEN delta="

    All functions use `from statistics import mean, quantiles, stdev` and `from wanctl.tuning.models import SafetyBounds, TuningResult`.

    REFACTOR: Extract any shared logic between the two calibrate functions if warranted. Ensure both match the StrategyFn signature: Callable[[list[dict], float, SafetyBounds, str], TuningResult | None].
