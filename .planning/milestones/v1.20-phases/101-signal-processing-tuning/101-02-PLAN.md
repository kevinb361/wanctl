<objective>
Wire signal processing strategies into the maintenance loop with layer-based round-robin,
extend the applier to handle signal processing parameters on SignalProcessor, and add
the layer rotation index to WANController. Critically, handle the load_time_constant_sec
to alpha_load conversion in the applier to avoid the clamp_to_step precision destruction
(Pitfall 3 from research).

Purpose: SIGP-04 requires bottom-up tuning (signal processing first, then EWMA, then thresholds)
with one layer per cycle. This prevents correlated parameter changes that would confound effect
isolation. The applier extension enables the new strategies to actually modify runtime parameters.

Output: Modified autorate_continuous.py with layer rotation, extended applier, and integration tests.
