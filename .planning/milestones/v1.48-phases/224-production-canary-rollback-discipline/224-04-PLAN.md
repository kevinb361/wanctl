<objective>
Run the canary observation window: sample /health + spine probe at a documented cadence over a
bounded ≥15-minute window; feed samples to the gate evaluator; produce verdict.json with a
kept-aligned, continue-observation, or rollback outcome. If a steady-state violation fires,
execute the Plan 01 rollback within the measured budget. Restart-window symptoms are NOT
rollback triggers (CANARY-02, CANARY-03).

Purpose: Phase 215 had a single end-of-window verdict. Phase 224 has a 15-minute window with
multiple samples because steady-state drift can appear gradually and the restart-window
symptom must be filtered out. The gate evaluator from Plan 02 is the source of truth.

Output: observation/ samples, verdict.json, and optionally rollback/ evidence.
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Observation samplers → /health + RouterOS | Read-only. Bound and loopback /health surfaces; REST/SSH router reads. |
| Gate-eval → sample JSON | Pure-function over JSON inputs. |
| Rollback → production host | Single mutation event under operator control; bounded to the snapshot restore + restart + verify sequence. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-224-15 | Tampering | Observation loop accidentally writes to host | mitigate | All commands are `curl` reads + `ssh ... curl` (read-only). No `sudo`, no `systemctl`. Task 1 grep-gate excludes mutation verbs. |
| T-224-16 | Denial of service | Sampling cadence overloads /health endpoint | accept | Default 60-second cadence at 9101 is well below the health endpoint's documented capacity; no risk at this rate. |
| T-224-17 | Elevation of privilege | Rollback skips verification step | mitigate | Task 3 requires operator-typed resume signal naming both rollback_duration AND post-revert /health.version. |
| T-224-18 | Repudiation | Verdict.json fabricated by operator preference | mitigate | Task 2 reads gate-eval output verbatim; no operator-editable fields. Verdict.json schema is checked in CI for shape. |
| T-224-19 | Information disclosure | Sample JSONs contain router credentials | mitigate | Samples come from /health (no credentials surfaced) and from spine-probe (redacted by Plan 02 design). |
| T-224-SC | Supply chain | New observation loop introduces packages | accept | Stdlib + bash + curl + jq. No new packages. |
</threat_model>
