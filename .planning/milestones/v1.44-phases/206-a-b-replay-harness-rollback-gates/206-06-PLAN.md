<objective>
Close verification gap G3 (shell wrapper exits rc=1 on missing option value instead of rc=2) by adding a `require_value()` helper that validates the next argument exists and is not another `--option` BEFORE any `shift 2`. Operator-input errors must classify as ABORT (rc=2), not BLOCK (rc=1) — BLOCK is reserved for threshold breaches with valid input.

**Why:** Phase 206 success criterion #4 and TOPO-05 require "fails closed on malformed/inconsistent input." The current wrapper uses `"${2:-}"; shift 2` which, under `set -e`, fails on shift if `$2` is missing, producing rc=1 — the operator cannot distinguish a real threshold breach from a typo'd CLI invocation. This plan adds explicit value validation.

This plan touches the shell wrapper ONLY (independent file from Plan 05's Python edits) — runs in Wave 2 (depends on 206-05 because both append to `tests/test_phase206_predeploy_gate.py`; running 06 in the same wave as 05 would race on the same file).

    ```bash
    # require_value OPT VALUE — abort with rc=2 if VALUE is empty or starts with `--`.
    # Call BEFORE any `shift 2` so a missing operator-input value classifies as ABORT,
    # not BLOCK (which is reserved for threshold breaches under valid input).
    require_value() {
        local opt="$1"
        local val="${2-}"
        if [[ -z "${val}" ]]; then
            log_abort "missing value for ${opt}"
            exit $EXIT_ABORT
        fi
        if [[ "${val}" == --* ]]; then
            log_abort "missing value for ${opt} (next token is option-like: ${val})"
            exit $EXIT_ABORT
        fi
    }
    ```

    ```bash
    while [[ $# -gt 0 ]]; do
        case "$1" in
            --baseline)
                require_value "--baseline" "${2-}"; BASELINE="$2"; shift 2 ;;
            --candidate)
                require_value "--candidate" "${2-}"; CANDIDATE="$2"; shift 2 ;;
            --soak-ndjson)

            --mode)
                require_value "--mode" "${2-}"; MODE="$2"; shift 2 ;;
            --ssh-target)
                require_value "--ssh-target" "${2-}"; SSH_TARGET="$2"; shift 2 ;;
            --window-start-iso8601)
                require_value "--window-start-iso8601" "${2-}"; WIN_START="$2"; shift 2 ;;
            --window-end-iso8601)
                require_value "--window-end-iso8601" "${2-}"; WIN_END="$2"; shift 2 ;;
            --window-hours)
                require_value "--window-hours" "${2-}"; WIN_HOURS="$2"; shift 2 ;;
            --restart-counter-start)
                require_value "--restart-counter-start" "${2-}"; RC_START="$2"; shift 2 ;;
            --restart-counter-end)
                require_value "--restart-counter-end" "${2-}"; RC_END="$2"; shift 2 ;;
            --journal-since)
                require_value "--journal-since" "${2-}"; JOURNAL_SINCE="$2"; shift 2 ;;
            -h|--help)
                usage; exit $EXIT_PASS ;;
            *)
                log_abort "unknown argument: $1"
                usage
                exit $EXIT_ABORT ;;
        esac
    done
    ```
