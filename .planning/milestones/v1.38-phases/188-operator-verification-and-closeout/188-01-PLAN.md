State boundaries (D-02, Phase 186):
- successful_count == 3 -> "healthy"
- successful_count == 2 -> "reduced"
- successful_count <= 1 -> "collapsed"  (count==1 is a single-reflector quorum, not trustworthy)

Phase 187 wiring (from 187-VERIFICATION.md):
- Zero-success cycles no longer present cached RTT as a healthy current measurement.
- measure_rtt() branches on BackgroundRTTThread.get_cycle_status() so the
  /health block reports current host truth on collapse (empty successful_hosts,
  state="collapsed") while SAFE-02 ICMP failure fallback is unchanged.
