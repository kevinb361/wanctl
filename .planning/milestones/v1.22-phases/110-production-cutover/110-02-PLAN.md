Via MikroTik REST API from any host that can reach 10.10.99.1:
```bash

# Disable ATT queues (use .id from output above, or name)

  'https://10.10.99.1/rest/queue/tree/WAN-Download-ATT' \

  'https://10.10.99.1/rest/queue/tree/WAN-Upload-ATT' \
  -d '{"disabled":"true"}'

```
