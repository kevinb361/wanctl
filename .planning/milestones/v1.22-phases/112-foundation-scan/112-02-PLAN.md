  <action>
Run all permission checks via SSH to cake-shaper VM (per D-08, D-09). All commands use `ssh kevin@10.10.110.223`.

1. Check /etc/wanctl/secrets permissions (FSCAN-04, expected: 0600, owner root):
   `ssh kevin@10.10.110.223 'sudo stat -c "%a %U:%G %n" /etc/wanctl/secrets'`

2. Check state directory permissions (expected: 0750):
   `ssh kevin@10.10.110.223 'sudo stat -c "%a %U:%G %n" /var/lib/wanctl/'`
   `ssh kevin@10.10.110.223 'sudo ls -la /var/lib/wanctl/'`

3. Check log directory permissions (expected: 0750):
   `ssh kevin@10.10.110.223 'sudo stat -c "%a %U:%G %n" /var/log/wanctl/'`
   `ssh kevin@10.10.110.223 'sudo ls -la /var/log/wanctl/'`

4. Check config file permissions (sensitive -- should not be world-readable):
   `ssh kevin@10.10.110.223 'sudo stat -c "%a %U:%G %n" /etc/wanctl/*.yaml'`

5. Check /opt/wanctl/ application permissions:
   `ssh kevin@10.10.110.223 'stat -c "%a %U:%G %n" /opt/wanctl/ /opt/wanctl/src/'`

If any permission is incorrect, document the finding with the current value and expected value. Do NOT fix permissions remotely -- document for human action.
