    2. Constants section (per CONVENTIONS.md section divider pattern):
       - UPLOAD_DEFAULTS: {"diffserv": "diffserv4", "split-gso": True, "ack-filter": True, "ingress": False, "ecn": False}
       - DOWNLOAD_DEFAULTS: {"diffserv": "diffserv4", "split-gso": True, "ack-filter": False, "ingress": True, "ecn": True}
       - DIRECTION_DEFAULTS: {"upload": UPLOAD_DEFAULTS, "download": DOWNLOAD_DEFAULTS}
       - TUNABLE_DEFAULTS: {"memlimit": "32mb", "rtt": "100ms"}
       - EXCLUDED_PARAMS: {"nat", "wash", "autorate-ingress"}
       - YAML_TO_TC_KEY: {"split_gso": "split-gso", "ack_filter": "ack-filter", "autorate_ingress": "autorate-ingress"}
       - VALID_OVERHEAD_KEYWORDS: {"docsis", "bridged-ptm", "ethernet", "pppoe-ptm", "bridged-llcsnap", "pppoa-vcmux", "pppoa-llc", "pppoe-vcmux", "pppoe-llcsnap", "conservative", "raw"}
       - OVERHEAD_READBACK: {"docsis": {"overhead": 18}, "bridged-ptm": {"overhead": 22}, "ethernet": {"overhead": 38}}
       - RTT_TO_MICROSECONDS: {"100ms": 100_000, "50ms": 50_000, "30ms": 30_000}
       - MEMLIMIT_TO_BYTES: {"32mb": 33_554_432, "16mb": 16_777_216, "64mb": 67_108_864}

    3. build_cake_params(direction, cake_config=None, bandwidth_kbit=None) -> dict[str, Any]:
       - Validate direction is in DIRECTION_DEFAULTS, raise ValueError if not
       - Start with copy of DIRECTION_DEFAULTS[direction]
       - Merge TUNABLE_DEFAULTS
       - If cake_config provided:
         - For each key/value: translate key via YAML_TO_TC_KEY, check against EXCLUDED_PARAMS (raise ConfigValidationError), set in params
       - Pop "overhead" from params, if it's a string: validate against VALID_OVERHEAD_KEYWORDS (raise ConfigValidationError if invalid), set as params["overhead_keyword"]
       - If bandwidth_kbit provided: set params["bandwidth"] = f"{bandwidth_kbit}kbit"
       - Return params

    4. build_expected_readback(params) -> dict[str, Any]:
       - If overhead_keyword present: look up in OVERHEAD_READBACK, update expected
       - If diffserv present: copy directly
       - If rtt present: convert string to microseconds (lookup or parse by stripping "ms" and multiplying by 1000)
       - If memlimit present: convert string to bytes (lookup or parse)
       - Return expected dict
