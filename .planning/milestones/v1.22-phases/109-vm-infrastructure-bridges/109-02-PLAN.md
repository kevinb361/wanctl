Present the user with NIC passthrough commands and interface discovery procedure.

**Step 1: Shut down VM and add passthrough NICs (per D-06)**
```bash
ssh root@10.10.110.124
qm shutdown 106
qm wait 106
qm set 106 --hostpci0 08:00.0
qm set 106 --hostpci1 09:00.0
qm set 106 --hostpci2 0c:00.0
qm set 106 --hostpci3 0c:00.1
qm config 106 | grep -E 'hostpci|net0'
```

**Step 2: Start VM and discover interface names (per D-07)**
```bash
qm start 106
sleep 15
ssh root@10.10.110.106 'ip -o link show | grep -v "lo:" | awk "{print \$2}" | tr -d ":"'
```

**Step 3: Map PCI addresses to interface names**
```bash
ssh root@10.10.110.106 'for iface in /sys/class/net/enp*; do name=$(basename "$iface"); pci=$(basename "$(readlink "$iface/device")"); echo "$name -> $pci"; done'
```

**Step 4: Verify NIC hardware to confirm mapping**
```bash
ssh root@10.10.110.106 'for iface in /sys/class/net/enp*; do name=$(basename "$iface"); pci=$(basename "$(readlink "$iface/device")"); vendor=$(cat "$iface/device/vendor" 2>/dev/null); device=$(cat "$iface/device/device" 2>/dev/null); echo "$name -> PCI $pci, ID $vendor:$device"; done'
```

Device IDs: 0x1533 = i210 (Spectrum pair), 0x1521 = i350 (ATT pair).

Wait for user to paste output and provide the NIC name mapping.
