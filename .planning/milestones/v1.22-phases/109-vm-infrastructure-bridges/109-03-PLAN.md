The user must SSH to cake-shaper (10.10.110.106) and execute these commands. In all commands below, replace SPEC_MODEM, SPEC_ROUTER, ATT_MODEM, ATT_ROUTER with the actual NIC names from the mapping table.

**Step 1: Create bridge .netdev files (per D-08, D-09)**
```bash
ssh root@10.10.110.106

cat > /etc/systemd/network/10-br-spectrum.netdev << 'EOF'
[NetDev]
Name=br-spectrum
Kind=bridge

[Bridge]
STP=false
ForwardDelaySec=0
EOF

cat > /etc/systemd/network/10-br-att.netdev << 'EOF'
[NetDev]
Name=br-att
Kind=bridge

[Bridge]
STP=false
ForwardDelaySec=0
EOF
```

**Step 2: Create member port .network files (per D-10, D-11)**
```bash
cat > /etc/systemd/network/20-spec-modem.network << EOF
[Match]
Name=SPEC_MODEM

[Network]
Bridge=br-spectrum

[Link]
RequiredForOnline=enslaved
EOF

cat > /etc/systemd/network/20-spec-router.network << EOF
[Match]
Name=SPEC_ROUTER

[Network]
Bridge=br-spectrum

[Link]
RequiredForOnline=enslaved
EOF

cat > /etc/systemd/network/20-att-modem.network << EOF
[Match]
Name=ATT_MODEM

[Network]
Bridge=br-att

[Link]
RequiredForOnline=enslaved
EOF

cat > /etc/systemd/network/20-att-router.network << EOF
[Match]
Name=ATT_ROUTER

[Network]
Bridge=br-att

[Link]
RequiredForOnline=enslaved
EOF
```

**Step 3: Create bridge .network files with no IP (per D-11)**
```bash
cat > /etc/systemd/network/30-br-spectrum.network << 'EOF'
[Match]
Name=br-spectrum

[Link]
RequiredForOnline=carrier
EOF

cat > /etc/systemd/network/30-br-att.network << 'EOF'
[Match]
Name=br-att

[Link]
RequiredForOnline=carrier
EOF
```

**Step 4: Create management interface .network file (per D-12)**
```bash
cat > /etc/systemd/network/40-ens18.network << 'EOF'
[Match]
Name=ens18

[Network]
Address=10.10.110.106/24
Gateway=10.10.110.1
DNS=10.10.110.1

[Link]
RequiredForOnline=routable
EOF
```

**Step 5: Switch from ifupdown to systemd-networkd**
```bash
systemctl enable systemd-networkd
systemctl enable systemd-networkd-wait-online
cp /etc/network/interfaces /etc/network/interfaces.bak
cat > /etc/network/interfaces << 'EOF'
auto lo
iface lo inet loopback
EOF
```

**Step 6: Verify files and reboot**
```bash
ls -la /etc/systemd/network/
reboot
```

After reboot (wait 30-60 seconds):
```bash
ssh root@10.10.110.106 'echo "SSH OK"'
```

echo ""
echo "=== Forward Delay (expect 0) ==="
echo "br-spectrum: $(cat /sys/class/net/br-spectrum/bridge/forward_delay)"

ip addr show br-spectrum | grep inet || echo "br-spectrum: no IP (correct)"

echo "=== Management IP ==="
ip addr show ens18 | grep "inet "

echo ""
echo "=== wait-online status ==="
systemctl status systemd-networkd-wait-online --no-pager | head -5
