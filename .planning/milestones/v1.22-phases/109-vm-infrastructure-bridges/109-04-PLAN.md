Present the user with the deployment procedure for cake-shaper. The user must run these commands from their workstation and on the VM.

**Step 1: Install Python dependencies on cake-shaper (per D-18)**
```bash
ssh root@10.10.110.106
apt-get update
apt-get install -y python3-pip python3-dev gcc rsync
pip3 install --break-system-packages icmplib requests paramiko pyyaml tabulate cryptography pexpect
python3 -c "import icmplib, requests, paramiko, yaml, tabulate; print('All deps OK')"
exit
```

**Step 2: Run install.sh to create wanctl user and directories**
From workstation:
```bash
cd ~/projects/wanctl
./scripts/deploy.sh --install-only 10.10.110.106
```

**Step 3: Deploy wanctl code via rsync**
```bash
rsync -av --delete --exclude=__pycache__ --exclude='*.pyc' \
  --rsync-path='sudo rsync' --chmod=F644,D755 --chown=root:root \
  src/wanctl/ root@10.10.110.106:/opt/wanctl/
```

**Step 4: Install systemd service with networkd ordering (per D-15)**
```bash
scp systemd/wanctl@.service root@10.10.110.106:/etc/systemd/system/
ssh root@10.10.110.106 'sed -i "s/After=network-online.target/After=network-online.target systemd-networkd-wait-online.service/" /etc/systemd/system/wanctl@.service && systemctl daemon-reload'
```

**Step 5: Install iproute2 and verify**
```bash
ssh root@10.10.110.106 'apt-get install -y iproute2 && tc -V'
```

**Step 6: Verify deployment**
```bash
ssh root@10.10.110.106 'ls /opt/wanctl/autorate_continuous.py && id wanctl && ls -la /var/lib/wanctl/ /var/log/wanctl/ && grep After /etc/systemd/system/wanctl@.service'
```

Present the user with CAKE test commands. Replace SPEC_ROUTER and ATT_ROUTER with actual names from the NIC mapping table.

**Step 1: Load CAKE module and test initialization**
```bash
ssh root@10.10.110.106

modprobe sch_cake

# Test CAKE on Spectrum router-side NIC (bandwidth is test value)
tc qdisc replace dev SPEC_ROUTER root cake bandwidth 500mbit diffserv4 split-gso overhead 18 mpu 64 memlimit 32mb rtt 100ms ingress ecn

# Test CAKE on ATT router-side NIC
tc qdisc replace dev ATT_ROUTER root cake bandwidth 50mbit diffserv4 split-gso overhead 0 mpu 64 memlimit 32mb rtt 100ms ack-filter

# Verify CAKE is attached (human-readable)
tc qdisc show dev SPEC_ROUTER
tc qdisc show dev ATT_ROUTER
```

**Step 2: Verify JSON stats collection**
```bash
tc -s -j qdisc show dev SPEC_ROUTER | python3 -c "import sys,json; data=json.load(sys.stdin); print('CAKE found:', any(q.get('kind')=='cake' for q in data))"
tc -s -j qdisc show dev ATT_ROUTER | python3 -c "import sys,json; data=json.load(sys.stdin); print('CAKE found:', any(q.get('kind')=='cake' for q in data))"
```

Expected: Both print "CAKE found: True"

**Step 3: Clean up test qdiscs**
```bash
tc qdisc del dev SPEC_ROUTER root
tc qdisc del dev ATT_ROUTER root
```

The wanctl daemon will re-create the qdiscs at startup via initialize_cake().
