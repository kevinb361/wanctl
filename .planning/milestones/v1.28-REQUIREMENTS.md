# Requirements: wanctl

**Defined:** 2026-04-04
**Core Value:** Sub-second congestion detection with 50ms control loops

## v1.28 Requirements

Requirements for v1.28 Infrastructure Optimization milestone. Each maps to roadmap phases.

### VM Optimization (VMOPT)

- [x] **VMOPT-01**: cake-shaper VM runs with 3 vCPUs, verified operational with balanced load across cores
- [x] **VMOPT-02**: NIC IRQ affinity configured so Spectrum and ATT bridge interrupts are distributed across available cores (eliminating 4.3x imbalance)
- [x] **VMOPT-03**: Kernel network sysctls tuned for bridge+CAKE workload (netdev_budget, netdev_max_backlog) with persistence across reboots
- [x] **VMOPT-04**: Download traffic classified into CAKE diffserv4 tins via nftables bridge forward rules, closing the bridge-before-router DSCP gap with persistence across reboots

### Router Optimization (RTOPT)

- [x] **RTOPT-01**: SFP+ interface uses multi-queue-ethernet-default queue type, eliminating TX queue drops during traffic bursts
- [x] **RTOPT-02**: Switch IRQ processing redistributed to reduce peak core utilization (cpu2 currently 30-46% while cpu1 at 5-9%)
- [x] **RTOPT-04**: WireGuard TX errors (821K over 2 weeks) diagnosed with root cause identified and fix applied

## Evidence Base

Data collected 2026-04-04 from live devices during RRUL testing session.

### cake-shaper VM (10.10.110.223)

- 2 vCPUs, 1.9 GB RAM, Debian 13
- Load average: 1.35-1.72 (near capacity on 2 cores even accounting for RRUL load)
- CPU breakdown: 47.6% softirq (bridge+CAKE), ~25% user (wanctl), 38% idle
- NIC IRQ imbalance: CPU0 handles 139M Spectrum interrupts, CPU1 handles 32M ATT (4.3x)
- CAKE on ens17 (Spectrum DL): 940Mbit, diffserv4, 7M drops/131M pkts shaped
- CAKE on ens28 (ATT DL): 63Mbit, diffserv4, 4 drops/8.5M pkts shaped

### RB5009 (10.10.99.1)

- RouterOS 7.20.7, CPU avg 29% (cpu2 peak 46%), 1,796 conntrack entries
- All 6 queue trees DISABLED (CAKE offloaded to cake-shaper VM)
- FastPath: no, FastTrack: no
- SFP+ tx-queue-drop: 404,196 (0.01% of 3.75B TX packets)
- Switch IRQ distribution: cpu2=527M, cpu3=547M, cpu0=737M, cpu1=548M
- WireGuard: tx-error=821,114, tx-drop=3,238 over 2w4d uptime
- 61 mangle rules, 43 firewall rules, 7 NAT rules, 9 RAW rules
