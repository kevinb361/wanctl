## Decision Thresholds (per D-07)

A config "wins" over baseline if it produces:
- 15%+ UL throughput improvement, OR
- 20%+ latency improvement (ICMP median or p99)

If NO config meets either threshold, keep current params (step_up=2, factor_down=0.85).

If MULTIPLE configs meet thresholds, prefer the one with:
1. Best UL throughput (primary -- this phase is about upload recovery)
2. Best latency (secondary -- always matters for bufferbloat)
3. No DL throughput regression >5% (safety check)
