Functions to implement:

1. `compute_stats(rtts: list[float]) -> dict | None` -- compute mean, median, stdev, min, max, p95, p99, count using `statistics` stdlib. Return None if len(rtts) < 2. Use `statistics.quantiles(rtts, n=100)` for percentiles (p95=index 94, p99=index 98).

2. `assess_jitter(stdev: float, wan_name: str) -> str` -- compare container stdev against `WAN_JITTER_REFERENCE[wan_name]["idle"]`. Return "NEGLIGIBLE (X.X% of WAN idle jitter)" if ratio < 0.10, else "NOTABLE (X.X% of WAN idle jitter)". Use default reference `{"idle": 0.5, "loaded": 3.0}` for unknown WAN names.

3. `measure_container(host: str, count: int = DEFAULT_COUNT, interval: float = DEFAULT_INTERVAL) -> dict | None` -- run `subprocess.run(["ping", "-c", str(count), "-i", str(interval), host], capture_output=True, text=True, timeout=count*interval+10)`. Parse with `parse_ping_output(result.stdout)`. Return `compute_stats(rtts)` with added "host" key, or None on TimeoutExpired/empty rtts.

4. `run_measurements(containers: dict, count: int, interval: float, runs: int) -> dict` -- for each container, run `measure_container` `runs` times, aggregate all RTT samples across runs, compute final stats. Return dict keyed by container name with stats dict values.

5. `capture_topology(container: str) -> dict` -- SSH to container, run `ip link show` and `ip addr show`. Return dict with keys `ip_link` and `ip_addr`. Handle `subprocess.TimeoutExpired` (return "timeout" values) and non-zero returncode (return "unavailable" values). Use `timeout=10` on subprocess.run.

6. `generate_report(results: dict, topology: dict, wan_mapping: dict | None = None) -> str` -- generate full markdown report. Default `wan_mapping = {"cake-spectrum": "spectrum", "cake-att": "att"}`. Sections:
   - `# Container Network Audit` header with date
   - `## Executive Summary` with overall pass/fail against 0.5ms threshold
   - `## Measurement Methodology` (count, interval, runs, host-to-container)
   - `## Per-Container Results` with markdown table per container: Mean, Median, P95, P99, Stddev, Min, Max, Samples
   - `## Jitter Analysis` with jitter assessment per container using `assess_jitter()`
   - `## Network Topology` with raw SSH output per container
   - `## Recommendation` based on whether mean < 0.5ms AND jitter is NEGLIGIBLE
   - Handle None stats for unreachable containers with "Container unreachable" message

7. `main()` -- argparse with:
   - `--count` (default 1000): samples per run
   - `--interval` (default 0.01): seconds between pings
   - `--runs` (default 5): number of measurement runs per container
   - `--output` (default "docs/CONTAINER_NETWORK_AUDIT.md"): output path
   - `--dry-run`: generate report from synthetic data (no network access)
   Print progress to stdout. Write report to output path. Print summary to stdout.
