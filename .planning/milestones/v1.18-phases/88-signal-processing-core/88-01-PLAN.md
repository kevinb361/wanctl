**Module-level constants:**
```python
MAD_SCALE_FACTOR = 1.4826  # 1 / Phi^-1(3/4) for Gaussian consistency
CYCLE_INTERVAL_SECONDS = 0.05  # 50ms -- matches autorate_continuous.py
```

**SignalResult frozen dataclass:**
```python
@dataclass(frozen=True, slots=True)
class SignalResult:
    filtered_rtt: float         # Post-Hampel RTT (or raw if warming up)
    raw_rtt: float              # Original measured RTT (always preserved)
    jitter_ms: float            # EWMA jitter from consecutive raw RTT deltas
    variance_ms2: float         # EWMA variance of raw RTT around load_rtt
    confidence: float           # 0-1 score: 1.0/(1.0 + variance/baseline^2)
    is_outlier: bool            # True if Hampel detected this sample as outlier
    outlier_rate: float         # Fraction of recent window that were outliers
    total_outliers: int         # Lifetime count of all outliers detected (per user decision)
    consecutive_outliers: int   # Current streak of consecutive outliers, resets on non-outlier (per user decision)
    warming_up: bool            # True if Hampel window not yet full
```

**SignalProcessor class:**
- `__init__(self, wan_name: str, config: dict, logger: logging.Logger)`:
  - Extract from config dict: `hampel_window_size` (default 7), `hampel_sigma_threshold` (default 3.0), `jitter_time_constant_sec` (default 2.0), `variance_time_constant_sec` (default 5.0)
  - Compute `_jitter_alpha = CYCLE_INTERVAL_SECONDS / jitter_time_constant_sec`
  - Compute `_variance_alpha = CYCLE_INTERVAL_SECONDS / variance_time_constant_sec`
  - Initialize `_window: deque[float] = deque(maxlen=window_size)`
  - Initialize `_outlier_window: deque[bool] = deque(maxlen=window_size)` for tracking outlier rate
  - Initialize `_jitter_ewma: float = 0.0`
  - Initialize `_variance_ewma: float = 0.0`
  - Initialize `_previous_raw_rtt: float | None = None`
  - Initialize `_total_outliers: int = 0` -- lifetime count, never resets
  - Initialize `_consecutive_outliers: int = 0` -- current streak, resets on non-outlier
  - Store wan_name and logger

- `process(self, raw_rtt: float, load_rtt: float, baseline_rtt: float) -> SignalResult`:
  1. **Warm-up check:** `warming_up = len(self._window) < self._window_size`
  2. **Hampel filter:**
     - If warming_up: `filtered_rtt = raw_rtt`, `is_outlier = False`
     - Else: call `_hampel_check(raw_rtt)` -> `(is_outlier, filtered_rtt)`
     - If is_outlier: log at INFO: `"{wan_name}: RTT outlier detected: {raw_rtt:.1f}ms -> {filtered_rtt:.1f}ms (outlier_rate={rate:.1%})"`
  3. **Add to window:** `self._window.append(raw_rtt)` (always raw, not filtered)
  4. **Track outlier:** `self._outlier_window.append(is_outlier)`
     - If is_outlier: `self._total_outliers += 1`; `self._consecutive_outliers += 1`
     - Else: `self._consecutive_outliers = 0`
  5. **Outlier rate:** `outlier_rate = sum(self._outlier_window) / len(self._outlier_window)` (safe: len always >= 1 after append)
  6. **Jitter:** call `_update_jitter(raw_rtt)` -> `jitter_ms`
  7. **Variance:** call `_update_variance(raw_rtt, load_rtt)` -> `variance_ms2`
  8. **Confidence:** call `_compute_confidence(variance_ms2, baseline_rtt)` -> `confidence`
  9. **Debug log:** `logger.debug(f"{wan_name}: signal rtt={raw_rtt:.1f}ms filtered={filtered_rtt:.1f}ms jitter={jitter_ms:.2f}ms var={variance_ms2:.2f} conf={confidence:.3f}")`
  10. Return `SignalResult(filtered_rtt=filtered_rtt, raw_rtt=raw_rtt, jitter_ms=jitter_ms, variance_ms2=variance_ms2, confidence=confidence, is_outlier=is_outlier, outlier_rate=outlier_rate, total_outliers=self._total_outliers, consecutive_outliers=self._consecutive_outliers, warming_up=warming_up)`

- `_hampel_check(self, new_value: float) -> tuple[bool, float]`:
  - `values = list(self._window)`
  - `med = statistics.median(values)`
  - `abs_devs = [abs(v - med) for v in values]`
  - `mad = statistics.median(abs_devs)`
  - `threshold = mad * MAD_SCALE_FACTOR * self._sigma_threshold`
  - If `threshold == 0.0`: return `(False, new_value)` -- MAD=0 guard
  - If `abs(new_value - med) > threshold`: return `(True, med)` -- outlier replaced
  - Else: return `(False, new_value)` -- not an outlier

- `_update_jitter(self, raw_rtt: float) -> float`:
  - If `self._previous_raw_rtt is None`: set `self._previous_raw_rtt = raw_rtt`, return `0.0`
  - `delta = abs(raw_rtt - self._previous_raw_rtt)`
  - `self._previous_raw_rtt = raw_rtt`
  - If `self._jitter_ewma == 0.0`: `self._jitter_ewma = delta` (first-sample init)
  - Else: `self._jitter_ewma = (1.0 - self._jitter_alpha) * self._jitter_ewma + self._jitter_alpha * delta`
  - Return `self._jitter_ewma`

- `_update_variance(self, raw_rtt: float, load_rtt: float) -> float`:
  - `deviation_sq = (raw_rtt - load_rtt) ** 2`
  - If `self._variance_ewma == 0.0`: `self._variance_ewma = deviation_sq` (first-sample init)
  - Else: `self._variance_ewma = (1.0 - self._variance_alpha) * self._variance_ewma + self._variance_alpha * deviation_sq`
  - Return `self._variance_ewma`

- `_compute_confidence(self, variance_ms2: float, baseline_rtt: float) -> float` (static-compatible):
  - If `baseline_rtt <= 0.0`: return `1.0`
  - Return `1.0 / (1.0 + variance_ms2 / (baseline_rtt ** 2))`
