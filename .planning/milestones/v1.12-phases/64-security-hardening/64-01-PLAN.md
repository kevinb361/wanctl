  <action>
1. In routeros_rest.py `__init__`, REMOVE the `urllib3.disable_warnings(InsecureRequestWarning)` call (lines 107-110).

2. Add a `_suppress_ssl_warnings` flag: `self._suppress_ssl_warnings = not verify_ssl`.

3. Override the session's request behavior by wrapping each HTTP call with a context manager. Create a private helper method `_request(method, url, **kwargs)` that:
   ```python
   def _request(self, method: str, url: str, **kwargs) -> requests.Response:
       if self._suppress_ssl_warnings:
           import warnings
           with warnings.catch_warnings():
               warnings.filterwarnings("ignore", category=InsecureRequestWarning)
               return self._session.request(method, url, **kwargs)
       return self._session.request(method, url, **kwargs)
   ```

4. Replace all `self._session.get(...)`, `self._session.patch(...)`, `self._session.post(...)` calls with `self._request("GET", ...)`, `self._request("PATCH", ...)`, `self._request("POST", ...)` respectively. There are ~8 call sites:

   - Line 397: `self._session.post(url, json={".id": queue_id}, timeout=timeout_val)`
   - Line 434: `self._session.get(url, params=params, timeout=timeout_val)`
   - Line 483: `self._session.patch(url, json={"disabled": disabled}, timeout=timeout_val)`
   - Line 531: `self._session.get(url, params={filter_key: filter_value}, timeout=timeout_val)`
   - Line 613: `self._session.patch(url, ...)`
   - Line 640: `self._session.get(url, params={"name": queue_name}, timeout=self.timeout)`
   - Line 660: `self._session.get(f"{self.base_url}/system/resource", timeout=5)`

5. Also remove the top-level import `from urllib3.exceptions import InsecureRequestWarning` (line 45) and move it inside the `_request` method where it's needed (lazy import only when suppression is active).
